隐藏Root常见问题&解决办法

发表于2023-02-05|更新于2023-02-06|Android玩机记录
|字数总计:782|阅读时长:2分钟|阅读量:

momo 相关

Pc 可以ctrl+f快速搜索自己需要的条目

Q : 可执行程序 su
　找到 magisk
A : 使用shamiko模块，在面具排除列表勾选该应用所有进程并重启 momo 即可。

Q : 仍然出现找到 magisk
A : 检查是否装了全局模块如字体，驱动等

Q : 找到 zygisk
A : 使用shamiko v0.6（126）模块即可，如果仍然出现关掉所有的 lsposed 模块，再次打开 momo 检查，如果没有找到zygisk就一个个打开排除。

不能在 lsposed 作用域勾选 momo，否则会出现到xposed框架&发现代码注入。

Q : 包管理服务异常
A : 关闭核心破解的禁用软件包管理器签名验证

Q : 数据未加密，挂载参数被修改
A : 系统为 data 默认未加密的官改包，官方都是强制启用加密功能，更换 rom 或者加密，一般不会出现。

Q : 设备正在运行非原厂系统
A : 删除system/addon.d这个文件夹

Q : 分区挂载异常
A : 不用管 /doge

Q : bootload 未锁定
A : 先备份 persist 分区/dev/block/by-name/persist，然后删除/mnt/vendor/persist/data文件夹就无法检测 bl 锁真实状态了，同时会出现TEE损坏。

Q : TEE 损坏
A : 使用了伪装上锁模块，大部分厂家（小米不会）解锁那一刻就会出现这个，如果想修复只能用原机的 dev/block/by-name/persist 的备份刷入
ps：或许可以

Q : 已开启调试模式
A : 开发者选项里关闭USB调试
ps：一般没软件会查这个目前我知道的就某不知名 fps 游戏未来之役

Q : 存在 Magisk 或 TWRP 等可疑文件
A : 因为你刷了第三方 recovery,TWRP，TWRP 会在你手机的内部存储生成了一个名叫TWRP的文件夹。解决方法：把手机内部存储存在的 TWRP 文件夹重命名，比如重命名为 TWRP1 就好了。

Q : 非 SDK 接口的限制失效
A : 开发者选项里打开USB调试、USB调试（安全设置），adb 运行以下命令来重置系统`非 SDK 接口限制

无需 root 权限运行，运行完不用重启立即生效

应用相关

一般来说，如果你按照momo相关教程走完后 momo 只检测到bootload未锁定基本上绝大部分软件都不会有问题了，下文主要讲解一些特定应用的疑难杂症。

Q : 中国电信 v10.0.1 更新后卡第一屏
A : 设置 magisk 更新通道为Canary并更新至最新版，在设置选择隐藏即可。

Q : 未来之役打开就跳转浏览器无法使用
A : 在 momo 只检测到bootload未锁定情况下安装 magiskv25.2 并随机包名安装即可
