MySQL for Windows安装教程

发表于2023-01-20|更新于2023-02-09|开源项目食用指南
|字数总计:737|阅读时长:2分钟|阅读量:

MySQL 是目前最为流行的开放源码的数据库，是完全网络化的跨平台的关系型数据库系统，它是由瑞典 MySQLAB 公司开发，目前属于Oracle公司。任何人都能从 Internet 下载 MySQL 软件，而无需支付任费用，并且“开放源码”意味着任何人都可以使用和修改该软件。

安装器安装

下载地址：https://dev.mysql.com/downloads/installer/

mysql 官网上提供了两种安装方式，第一种是在线版联网安装，第二种是本地安装。二者的区别是前者是联网安装，当安装时必须能访问互联网，后者是离线安装使用的，一般建议下载离线安装使用的版本。

以上两种安装方法均为图形界面向导方式安装，优点是可以比较清晰地看到整个 mysql 安装过程，并且可以选择性的安装所需的功能。缺点是安装过程中会出现一些环境依赖问题，导致安装失败。
